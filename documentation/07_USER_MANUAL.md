# Noor-e-Hidayat: User Manual

## 1. Authentication & Roles
The app comes pre-seeded with two accounts for testing:

### **Imam Account** (Full Access)
- **Email**: `imam@noor.com`
- **Capabilities**: Write Khutbahs, View Dashboard Analytics.

### **Admin Account** (System Control)
- **Email**: `admin@noor.com`
- **Capabilities**: Verify Masjids, Manage Global Content.

### **Guest Mode** (Default)
- **Capabilities**: Read Quran, Search Masjids, Read Hadith, Ask AI.

## 2. Features Guide

### Masjid Network
1. Navigate to **Masjid Network**.
2. **Guest**: Browse the list of Masjids.
3. **Logged In**: Click **"+ Register Masjid"** to add a new entry. It will persist even if you refresh the page.

### Khutbah Builder
1. Login as `imam@noor.com`.
2. Navigate to **Khutbah Builder**.
3. Use the AI Copilot to generate an outline.
4. **Save Draft**: Your work is saved locally and will appear in the Dashboard.

### Hadith Library
1. Navigate to **Hadith Library**.
2. Select a book (e.g., Sahih Bukhari).
3. Read translations in English, Urdu, and Arabic.
